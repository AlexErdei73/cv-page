  • Learning web development through the Odin Project / January 2020 – present

In the foundation course I learned the basics of web development. This course took me from setting up my local development machine with Linux or macOS operational system to building a simple calculator on a website with Java Script, HTML and CSS. I learned there the fundamental concepts of git, GitHub, text editors (VS Code), and the goal of coding languages in web development. The main emphasize is to give anybody a comprehensive basic knowledge about the main front-end coding languages with teaching good coding practices in Java Script. It also teaches how to debug code, how to ask good questions and all this with learn by doing in a project based manner. The main content of this module can be found here. I also did a pair programming project with another learner, but this project has been removed from the curriculum since that.

After the foundations I learned Java Script on advanced level with creating several projects. These projects dealt with intermediate and advanced topics and the learning culminated into a final project, which in my case was a simplified Facebook clone, which uses Firebase back-end. I have chosen to implement this by React, just like the original site was built. I also used Bootstrap, react-bootstrap and reactfire to connect the React app with Firebase. The roadmap, which leads here is containing topics like the following:

  • OOP in Java Script (constructor function syntax, factory functions and the module pattern, ES6 classes) with strong emphasis given to the SOLID principles, especially to the single responsibility principle and the loose coupling of objects in OOP
  • Using webpack to set up a project with dependencies. ES6 modules.
  • Creating DOM elements and full webpages with Java Script
  • Using webpack to include assets in the bundle. Transpilers, source map.
  • Automated unit testing with Jest, TDD development of OOP code, pure functions and their importance>
  • Java script in practice, menus, forms, carousel components, etc.
  • Using web APIs, the REST API. Fetching data from an API. Promises, asynchronous programming in JS, avoiding call-back-hell with promises, JSON response format.
  • React basics up to function components and the useState and useEffect hooks
  • The React router
  • Data driven functional programming in React, the principle of the single source of truth
  • Using Firebase as a back-end, Firebase firestore, Firebase storage, Firebase auth and cloud functions

The full list of topics and projects can be found here.

I also plan to deal with topics, which the Odin project does not supposed to teach, like for example:

  • I want to learn how to test front-end React code up to end to end testing. Some companies do not test their front-end code by automated test tools, they only do visual testing. I know it is a relatively difficult topic, because there are tools, which are only built for this purpose. I would like to learn how to use these on the simplest way and what are the real advantages of automated front-end code testing.
  • Mobile devices are more and more important, so I would like to get an insight into mobile development through React Native.
  • I would also like to learn the newest React tools, like the context API for example.
  • I have seen that a state management system like Redux can simplify React code, because some other learners used it to their final project. I would like to know how it can simplify my React code.
