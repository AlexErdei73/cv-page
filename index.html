My dev website
https://github.com/AlexErdei73/my-dev-website

This is my blogging website, where other developers can post their blogs after registration or like the blog posts. The API is a REST API, which I built with Node and Express. The API is protected with JSON Web Tokens. I used passport for the authentication with a passport-jwt strategy. I used the jsonwebtoken npm package to digitally sign the tokens. The API also uses the cors library to acomplish CORS. The emphasize is on good security practices end error handling. Deployment is Heroku. The front-end is a React app, created with CRA and deployed on GitHub pages.
